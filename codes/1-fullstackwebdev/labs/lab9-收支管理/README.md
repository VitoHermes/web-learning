# 使用了 useContext 
## 使用 useContext 的目的是为了在组件之间共享数据
## 使用 useContext 的步骤是：
1. 创建一个 Context 对象
2. 使用 Provider 组件包裹需要共享数据的组件
3. 在需要使用共享数据的组件中使用 useContext 钩子函数
