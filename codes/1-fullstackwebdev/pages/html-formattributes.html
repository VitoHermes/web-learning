HTML Form Attributes

Attribute Description Example
action Specifies the URL of the page that will process the form data when the form is submitted <form action="/action_page.php">
method Specifies the HTTP method to use when submitting the form <form method="post">
target Specifies where to display the response that is received after submitting the form <form target="_blank">
autocomplete Specifies whether a form should have autocomplete on or off <form autocomplete="on">
novalidate Specifies that the form should not be validated when submitted <form novalidate>
