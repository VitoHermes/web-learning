HTML Attributes

Tag Description
<href> The href attribute specifies the URL of the page the link goes to
<src> The src attribute specifies the path to the image to display(absolute or relative)
<width and height> The width and height attributes specify the size of the image
<alt> The alt attribute specifies an alternate text for an image, if the image cannot be displayed
<style> The style attribute specifies a CSS style for an element
<lang> The lang attribute specifies the language of the element's content
<title> The title attribute specifies a title for the document
