The HTML element defines emphasized text. The content inside is typically displayed in italic.

The HTML element defines text which is marked or highlighted for reference or notation purposes. The content inside is typically displayed in yellow.

The HTML element defines smaller text.

The HTML element defines text that has been deleted from a document. Browsers will usually strike through deleted text.

The HTML element defines a text that has been inserted into a document. Browsers will usually underline inserted text.

The HTML element defines subscripted text. Subscripted text appears half a character below the baseline of the main text, and is sometimes rendered in a smaller font.

The HTML element defines superscripted text. Superscripted text appears half a character above the baseline of the main text, and is sometimes rendered in a smaller font.
